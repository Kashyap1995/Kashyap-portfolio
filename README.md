# Kashyap Dave - QA Portfolio

A modern, professional portfolio website for a Senior QA Engineer built with React and Vite.

## Features

- 🎨 Modern dark theme with professional colors
- ✨ Smooth animations using Framer Motion
- 📱 Fully responsive design
- 🚀 Optimized for performance
- ♿ Accessible and SEO-friendly
- 🌐 Ready for Vercel deployment

## Sections

- **Home** - Hero section with introduction
- **About** - Professional background and highlights
- **Skills** - Technical skills and tools
- **Experience** - Work experience and achievements
- **Projects** - Portfolio of projects
- **Certifications** - Professional certifications
- **Contact** - Contact information and links

## Tech Stack

- React 18
- Vite
- Framer Motion (animations)
- Lucide React (icons)
- CSS3 (styling)

## Getting Started

### Prerequisites

- Node.js 16+ and npm/yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd Kashyap_Portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```

5. Preview production build
```bash
npm run preview
```

## Deployment

### Vercel

1. Push your code to GitHub
2. Import your repository on [Vercel](https://vercel.com)
3. Vercel will automatically detect Vite and deploy
4. Your site will be live!

### Manual Deployment

The `dist` folder contains the production build. Deploy it to any static hosting service.

## Customization

- Update personal information in component files
- Modify colors in `src/index.css` CSS variables
- Add/remove sections as needed
- Update links and contact information
